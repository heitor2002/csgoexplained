# Projeto CS:GO Explained

https://github.com/heitor2002/csgoexplained/blob/main/LICENSE

## Sobre o projeto

CS:GO Explained foi é uma aplicação front-end responsiva, sendo meu primeiro projeto, feito icinialmente para estudo tanto de novas tecnologias quanto do funcionamento de hospedagens de sites.

O projeto utiliza de referência o jogo Counter-Strike: Global Offensive, e explora diversas funcionalidades do jogo, ensinando posições de cada mapa, utilitários, configurações, etc.

## Front-end

- HTML5
- CSS
- JS
- Jquery

# Descrição do projeto

O objetivo do projeto foi ter um visual simples e intuitivo, dessa forma, o usuário pode usufruir da aplicação sem passar por qualquer dificuldade. Pelo fato do projeto ter sido resultado do estudo de novas tecnologias, utilizei Jquery para fazer animações simples de fade-in e fade-out.

## Explicando o uso

Na página inicial nos deparamos com um menu capaz de nos direcionar para 4 resultados diferentes:

- Funções: Página que explicará as funções de cada player no jogo e seu posicionamento.
- Utilitários: Ao clicar nesta você terá acesso a mais 3 novas opções, onde será possível escolher a respectiva granada desejada, onde após selecionada você é direcionado(a) para uma página de escolha de mapa. Ao escolher o mapa você irá se deparar com o desenho do escolhido, e neste haverá botões clicáveis que abrem uma aula ensinando a como fazer tal utilitário.
- Mapas: Página que disponibiliza diversas opções de mapas para que você aprenda cada posição no respectivo mapa.
- Comandos: Página que apresenta algumas configurações de console no jogo, tanto para configurar separadamente comandos que a tela do jogo não disponibiliza quanto para adequar ao gosto do jogador.
